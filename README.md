
# Gujarat_Vidyapith Repository

This repository contains academic projects, assignments, and practical work completed as part of coursework at **Gujarat Vidyapith**.

The purpose of this repository is to organize and maintain programming exercises, project files, and learning materials in one place. Many academic programs encourage storing project code in repositories for future reference and reuse. ([gujaratvidyapith.org][1])

## Repository Structure

The repository contains multiple folders related to different subjects and projects.


## Projects Included

### 1. FleetFlow (Fleet Management System)

FleetFlow is a fleet management system designed to manage:

* Vehicles
* Trips
* Drivers
* Maintenance logs
* Expenses
* Analytics

### Modules

1. Main Dashboard
2. Vehicle Registry (Asset Management)
3. Trip Dispatcher & Management
4. Maintenance & Service Logs
5. Expense & Fuel Logging
6. Driver Performance & Safety Profiles
7. Operational Analytics & Financial Reports
